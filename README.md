# Campus Data

Campus Data is a Nigerian VTU platform designed to help students buy affordable data, airtime, and other digital services quickly and securely.

## Features
- User Registration & Login
- Buy Data
- Buy Airtime
- Wallet Funding
- Transaction History
- Referral System
- Admin Dashboard

## Tech Stack
- React
- Tailwind CSS
- Node.js
- Express.js
- PostgreSQL
- React Native
- Paystack API
- VTU API
